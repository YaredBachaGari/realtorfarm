# Deep Research Agent

Purpose: only after deterministic scoring finds an outreach-qualified lead, research owner/property context.

Inputs: one PropertyLead with evidence.
Outputs: concise notes with citations: ownership, related parcels, probate/case details, lien stack, tax status, mailing address mismatch, and confidence caveats.

Token policy: run this only for qualified leads, never for Tier 4-only context rows.
